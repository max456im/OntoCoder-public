# Архитектурный обзор OntoCoder Public 2.0

---

## Слои системы

```
┌───────────────────────────────┐
│        Онтоформа (.onto)      │ ← Юридико-онтологический артефакт
└───────────────┬───────────────┘
                ▼
┌───────────────────────────────┐
│        ontoc (компилятор)     │ ← Валидация AENGA/SGCL/CLA
└───────────────┬───────────────┘
                ▼
┌───────────────────────────────┐
│    onto-runtime (среда)       │ ← SGRL + NoemaSlow/NoemaFast
│  + shadow-switching monitor   │
└───────────────┬───────────────┘
                ▼
┌───────────────────────────────┐
│ ontoreg (реестр форм)         │ ← Хранит CLA-метаданные
└───────────────────────────────┘
```

---

## Ключевые инварианты

| Инвариант | Гарантия |
|---------|--------|
| **AENGA загружается первым** | Этическое ядро инициализируется до любого модуля |
| **SGRL неотключаем** | Даже root не может отключить shadow monitor |
| **onto16 → onto8 только через VMA** | Сохранение когнитивной автономии |
| **CLA привязан к данным** | Профиль пользователя содержит `jurisdiction_tag` |

---

## Модель данных

- **onto8**: внутренние состояния (рациональные/иррациональные)  
- **onto16**: профиль с социальным контекстом (zodiac, MBTI, proximity)  
- **onto-form**: декларация системы (публичная, верифицируемая)  

> Все переходы между уровнями **контролируются AENGA**.
